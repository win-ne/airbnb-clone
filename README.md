# Airbnb clone built with Next.js, Strapi, and v0
1. Requirements:
- Node.js
- Yarn
- Turborepo
2. Install dependencies:
```sh
yarn install
```
3. Run the project:
```sh
turbo dev
```
1. Add host and listing content.
2. Enable create, find, and findOne actions for all content and media types on Strapi admin panel.
3. Check out the clone at http://localhost:3000/.
